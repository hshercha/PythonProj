About:
This program takes in a file that consists of each individual customer information( name, userID, latitude, longitude ) as a json string.
It then parses those individual json string and retrieves the customer information.
After it retrieves the information, it displays the list of the customers who lives within 100km range of a certain latitude and longitude.

To run the main script:
    	>cd Source
	>python Invitation.py [fileName]

To run all tests:

   	 >python test_all.py
